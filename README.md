# Team 12 Portfolio - Innovation & Excellence

A modern, professional portfolio website showcasing Team 12's expertise in web development, innovation, and collaboration. Built with clean HTML, CSS, and JavaScript to deliver an exceptional user experience.

## 🌟 Live Demo

Visit our portfolio: **[https://suganrajtech-jpg.github.io/NM-PORTFOLIO-/](https://suganrajtech-jpg.github.io/NM-PORTFOLIO-/)**

## 🚀 Features

### Core Functionality
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern Navigation**: Fixed header with smooth scrolling and mobile hamburger menu
- **Interactive Elements**: Hover effects, smooth transitions, and micro-interactions
- **Contact Form**: Functional contact form with validation and success feedback

### Pages & Sections
- **Home Page**: Hero section with call-to-action and feature highlights
- **About Page**: Team mission, values, and impressive statistics
- **Contact Page**: Contact form and comprehensive contact information

### Design Highlights
- **Dark Theme**: Professional dark color scheme with purple and cyan accents
- **Glass Morphism**: Modern glass effects with backdrop blur
- **Gradient Text**: Eye-catching gradient text effects
- **SVG Icons**: Scalable vector icons throughout the site
- **Typography**: Clean, modern typography using system fonts

## 🛠️ Technologies Used

### Frontend Stack
- **HTML5**: Semantic markup with proper meta tags and accessibility
- **CSS3**: 
  - CSS Variables for theming
  - Flexbox and Grid layouts
  - CSS animations and transitions
  - Backdrop filters for glass effects
- **Vanilla JavaScript**: 
  - ES6+ features
  - Event handling
  - Form validation
  - Mobile menu functionality

### Design System
- **Color Palette**: 
  - Primary: Deep blue (#1f203e)
  - Accent: Purple (#8b5cf6) and Cyan (#00d9ff)
  - Text: Light grays (#f7f9fb, #a8b3cf)
- **Typography**: System font stack for optimal performance
- **Spacing**: Consistent spacing using CSS variables

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software required

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/suganrajtech-jpg/NM-PORTFOLIO-.git
   ```

2. Navigate to the project directory:
   ```bash
   cd NM-PORTFOLIO-
   ```

3. Open `index.html` in your web browser:
   ```bash
   # Simply double-click the file or use:
   open index.html
   ```

## 🔧 Customization

### Updating Content
- **Team Information**: Edit text content in HTML files
- **Contact Details**: Update contact.html with your information
- **Colors**: Modify CSS variables in styles.css
- **Images**: Replace hero-bg.jpg with your own background

### Adding New Pages
1. Create new HTML file following the existing structure
2. Include navigation and consistent styling
3. Add link to navigation menu in all HTML files
4. Update script.js if additional functionality needed

### Modifying Styles
- **Colors**: Update CSS variables in `:root`
- **Typography**: Modify font families in `body` selector
- **Layout**: Adjust container max-width and padding
- **Animations**: Update transition durations and effects
